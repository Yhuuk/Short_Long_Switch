## 通用临时文件忽略规则
| 规则       | 文件类型       | 说明                      |
| -------- | ---------- | ----------------------- |
| `*.bak`  | 备份文件       | 有些编辑器或工具修改文件前会生成备份      |
| `*.tmp`  | 临时文件       | 软件运行中产生的临时缓存            |
| `*.log`  | 日志文件       | 编译、运行、调试产生的日志           |
| `*.orig` | 冲突/合并原始文件  | Git 合并冲突或 patch 失败时可能产生 |
| `*.swp`  | Vim 临时交换文件 | 用 Vim 编辑时可能出现           |

## 电路引脚说明
- 第一版
1. `PA9` 是电源控制引脚，通过三极管实现电源自保持。高电平打开，低电平关闭 power_control_Pin
2. `PA15` 是按键检测引脚，默认为上拉输入，外部也有上拉电路。检测按键按下时是否为低电平 Key_Detectd_Pin
3. `PB5` 是板子上的板载LED，用于前期调试阶段的测试，低电平点亮LED
4. `PB7` 是MOS栅极驱动的控制引脚，高电平开启，低电平关闭，默认为低电平关闭 Mos_drived_Pin
5. `PB11` 是按键自带的LED灯，高电平点亮，低电平关闭 Key_LED_Pin

- 第二版
1. `PA10` 是电源控制引脚，通过三极管实现电源自保持。高电平打开，低电平关闭
2. `PA8` 是按键检测引脚，默认为上拉输入，外部也有上拉电路。检测按键按下时是否为低电平
~~3. `PB5` 是板子上的板载LED，用于前期调试阶段的测试，低电平点亮LED~~
4. `PA1` 是MOS栅极驱动的控制引脚，高电平开启，低电平关闭，默认为低电平关闭
5. `PA9` 是按键自带的LED灯，高电平点亮，低电平关闭
> **说明**: 第二版的改动就是把对应功能的引脚改成其它的 `IO` 口，然后就是第一版板上的 `LED` 是作为测试用的，现在这个板载 `LED` 就舍弃不用了
