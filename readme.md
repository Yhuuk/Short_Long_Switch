# 短按+长按 开关机

## 通用临时文件忽略规则
| 规则       | 文件类型       | 说明                      |
| -------- | ---------- | ----------------------- |
| `*.bak`  | 备份文件       | 有些编辑器或工具修改文件前会生成备份      |
| `*.tmp`  | 临时文件       | 软件运行中产生的临时缓存            |
| `*.log`  | 日志文件       | 编译、运行、调试产生的日志           |
| `*.orig` | 冲突/合并原始文件  | Git 合并冲突或 patch 失败时可能产生 |
| `*.swp`  | Vim 临时交换文件 | 用 Vim 编辑时可能出现           |

## 电路引脚说明
1. `PA9` 是电源控制引脚，通过三极管实现电源自保持。高电平打开，低电平关闭
2. `PA15` 是按键检测引脚，默认为上拉输入，外部也有上拉电路。检测按键按下时是否为低电平
3. `PB5` 是板子上的板载LED，用于前期调试阶段的测试，低电平点亮LED
4. `PB7` 是MOS栅极驱动的控制引脚，高电平开启，低电平关闭，默认为低电平关闭
5. `PB11` 是按键自带的LED灯，高电平点亮，低电平关闭
